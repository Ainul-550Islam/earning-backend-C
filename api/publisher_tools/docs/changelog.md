# Changelog

## v2.0.0 — Publisher Tools Complete System

### New Features
- **A/B Testing System**: Complete statistical testing for placements, floor prices, ad formats
- **Header Bidding Support**: Prebid.js integration with 20+ bidders
- **Hybrid Mediation**: Combine header bidding + waterfall for maximum revenue
- **Advanced Fraud Detection**: Multi-layer IVT detection (bot, click fraud, device farm, geo validation)
- **Webhook System**: Real-time event notifications for all publisher events
- **KYC System**: Full identity verification workflow with document management
- **Subscription Plans**: Free/Standard/Premium/Enterprise tier system
- **Publisher Referrals**: Multi-level referral commission system
- **Publisher Brand/Store**: Public profile pages and white-label support
- **Performance Leaderboard**: Monthly publisher rankings
- **Payment Disputes**: Formal dispute resolution workflow

### Analytics Improvements
- Real-time dashboard with live stats
- Revenue forecasting (linear regression)
- RPM analysis by site and category
- CTR anomaly detection
- Hourly/daily/weekly/monthly reporting
- Custom report builder with dimension/metric selection
- CSV export support

### Ad Network Integrations Added
- AdMob, Facebook Audience Network, AppLovin
- Unity Ads, IronSource, Vungle
- Chartboost, Tapjoy, Fyber
- Google Ad Manager, Prebid, Amazon Publisher Services
- Verizon Media, Rubicon, Index Exchange
- OpenX, PubMatic, Sovrn, TripleLift

### Optimization Tools
- Yield optimizer with floor price suggestions
- Ad refresh rate optimizer
- Ad load analyzer
- Frequency cap analyzer
- Viewability optimizer
- Latency optimizer
- Geo optimizer (high-value country targeting)
- Device-specific optimization
- Time-based optimization
- Audience segment optimization

### Payment Enhancements
- bKash, Nagad, Rocket support
- USDT (TRC-20) and Bitcoin support
- Withholding tax calculation by country
- W-8BEN form support
- Invoice HTML generation
- Payment milestone notifications

### API Additions
- 80+ REST endpoints
- Webhook delivery system with retry logic
- Bulk operations for sites, apps, ad units, placements
- Custom report builder API
- Real-time fraud detection API
