# Seller Guide

## Getting Started

1. **Register** — Create account at the marketplace
2. **Create Seller Profile** — `POST /api/marketplace/sellers/`
3. **Submit KYC** — Upload NID front/back + selfie
4. **Wait for Approval** — Admin reviews within 24-48 hours
5. **Add Products** — `POST /api/marketplace/products/`

## Commission Structure

| Category | Commission |
|----------|------------|
| Electronics | 12% + 5 BDT |
| Fashion | 10% + 5 BDT |
| Books | 8% + 5 BDT |
| Default | 10% + 5 BDT |

## Payout Schedule

- Escrow hold: **7 days** after delivery confirmation
- Minimum payout: **100 BDT**
- Methods: bKash, Nagad, Rocket, Bank Transfer

## Order Management

1. New order → confirm within **48 hours** (else auto-cancelled)
2. Pack item → hand over to courier
3. Enter tracking number in seller dashboard
4. Customer receives → escrow released after 7 days

## Tips for Higher Sales

- Complete your store profile with logo and description
- Add high-quality product images
- Respond to customer questions promptly
- Maintain a high rating (4.5+ recommended)
- Use promotional campaigns during Eid, New Year, etc.
