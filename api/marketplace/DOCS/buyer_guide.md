# Buyer Guide

## How to Order

1. Browse products by category or use search
2. Add items to cart
3. Apply coupon code (if available)
4. Proceed to checkout — enter delivery address
5. Choose payment method (bKash, Nagad, COD, etc.)
6. Confirm order

## Tracking Your Order

- After placing order, go to **My Orders**
- Click on an order to see real-time tracking
- Statuses: Pending → Confirmed → Shipped → Delivered

## Refund & Return Policy

- Request refund within **30 days** of delivery
- Provide photo evidence of issue
- Refunds processed within **3-5 business days**

## Coupon Usage

- Enter coupon code at checkout
- One coupon per order
- Cannot be combined with other offers
