# Changelog

All notable changes to Earning Platform are documented here.

## [1.0.0] - 2026-03-24

### Added
- 38 backend modules (ad_networks, wallet, fraud_detection, analytics, etc.)
- 48 frontend pages with dark cyberpunk UI
- Multi-tenant architecture
- API Endpoint Control — live enable/disable 3874 endpoints
- Payment gateways: bKash, Nagad, Stripe, PayPal, SSLCommerz
- Fraud detection with IP blacklisting
- Real-time analytics dashboard
- Celery async task queue
- Railway + Vercel deploy config
- Docker + docker-compose support
- GitHub Actions CI/CD pipeline
- JWT authentication with OTP
- WebSocket support via Django Channels
- AWS S3 file storage support
- Full test coverage for core modules
